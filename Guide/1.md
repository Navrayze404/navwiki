## Mount the Root Partition

Mount partisi root ke direktori sementara (`/mnt`). Seluruh file sistem Nav Linux akan diinstal ke lokasi ini.

```bash
mount /dev/sdb6 /mnt
```

---

## Create the EFI Mount Point

Buat direktori tempat EFI System Partition akan di-mount.

```bash
mkdir -p /mnt/boot/efi
```

Perintah ini aman dijalankan berulang kali. Jika direktori sudah ada, tidak akan terjadi kesalahan.

---

## Mount the EFI System Partition

Mount EFI System Partition ke dalam target instalasi.

```bash
mount /dev/sdb4 /mnt/boot/efi
```

Bootloader GRUB akan diinstal ke partisi ini agar sistem dapat melakukan boot setelah instalasi selesai.

> Jika Anda menggunakan EFI System Partition yang sudah ada, **jangan format partisi tersebut**.

---

## Enable the Swap Partition

Aktifkan partisi swap.

```bash
swapon /dev/sdb5
```

Swap digunakan sebagai memori virtual ketika RAM mulai penuh. Mengaktifkannya selama instalasi juga memastikan konfigurasi swap dapat langsung digunakan setelah sistem terpasang.

Verifikasi:

```bash
swapon --show
```
